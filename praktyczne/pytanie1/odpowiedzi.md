# Zadanie 1 – Kody Unicode emoji

| Emoji | Opis | Kod Unicode |
|-------|------|-------------|
| 🧀 | ser | U+1F9C0 |
| 💡 | żarówka | U+1F4A1 |
| 🧠 | mózg | U+1F9E0 |
| ✏️ | ołówek | U+270F |
| ✈️ | samolot | U+2708 |
| 🪣 | łopata | U+1FAA3 |

**Szczegóły:**
- 🧀 Ser (Cheese): **U+1F9C0**
- 💡 Żarówka (Light Bulb): **U+1F4A1**
- 🧠 Mózg (Brain): **U+1F9E0**
- ✏️ Ołówek (Pencil): **U+270F** (+ wariant U+FE0F)
- ✈️ Samolot (Airplane): **U+2708** (+ wariant U+FE0F)
- 🪣 Łopata (Bucket/Spade): **U+1FAA3**

> Niektóre emoji mogą mieć dodatkowy kod wariantu U+FE0F, który sprawia, że wyświetlają się jako kolorowe emoji zamiast symboli tekstowych.
